# Rajnish Kumar Sharma Portfolio

A responsive single-page portfolio website built with HTML5, CSS3, and JavaScript to showcase Rajnish Kumar Sharma's profile, services, selected projects, and contact information.

## Live Links

- Portfolio: [https://rajnissharma.github.io/My-Portfolio-/](https://rajnissharma.github.io/My-Portfolio-/)
- Repository: [https://github.com/RajnisSharma/My-Portfolio-](https://github.com/RajnisSharma/My-Portfolio-)

## About The Project

This project is a static personal portfolio designed to present Rajnish Kumar Sharma as a Full Stack Python Developer. The website introduces his background, highlights the services he offers, showcases live frontend projects, and provides a simple way for visitors to get in touch.

The site is lightweight, easy to customize, and suitable for deployment on GitHub Pages or any other static hosting platform.

## Main Sections

- Home
- About Me
- Services
- Portfolio
- Contact
- Footer

## Features

- Responsive single-page layout for desktop, tablet, and mobile devices
- Fixed navigation bar with anchor links to all sections
- Mobile menu toggle for smaller screens
- Typing animation in the hero section using `Typed.js`
- Social profile links and email shortcut
- Download CV button
- Service cards with hover effects
- Portfolio project cards with descriptions and live links
- Client-side contact form interaction with a success alert
- Modern gradient styling, shadows, and transitions

## Getting Started

No package installation or build process is required.

1. Clone the repository:

```bash
git clone https://github.com/RajnisSharma/My-Portfolio-.git
```

2. Move into the project folder:

```bash
cd My-Portfolio-
```

3. Launch the website:

- Open `index.html` directly in your browser, or
- Use a local static server such as the VS Code Live Server extension

## Customization Guide

### Update Personal Information

Edit `index.html` to change:

- Name and introduction text
- About section content
- Services content
- Social media links
- Contact details
- Resume link

### Update Project Showcase

Edit the portfolio cards in `index.html` to:

- Add or remove projects
- Replace preview images
- Change descriptions
- Update live demo links

### Update Design

Edit `style.css` to customize:

- Colors
- Typography
- Layout spacing
- Button styles
- Section backgrounds
- Responsive breakpoints

### Update Interactive Behavior

Edit `script.js` to modify:

- Mobile navigation toggle behavior
- Typing animation strings and speed
- Contact form submission handling

## Important Notes

- The page directly references assets from the `Images/` folder. Keep that folder in place when deploying or sharing the project.
- Expected asset types include the profile image, about image, project thumbnails, site icon, and resume PDF.
- The contact form is currently frontend-only. It prevents default submission, shows an alert message, and resets the form.
- `Boxicons` and `Typed.js` are loaded through CDNs, so those resources need internet access unless you host them locally.
- Some links in the current HTML, such as Instagram and a few `Read More` buttons, are placeholders and can be updated later.

## Deployment

This project can be deployed easily on:

- GitHub Pages
- Netlify
- Vercel
- Firebase Hosting

For GitHub Pages:

1. Push the repository to GitHub.
2. Open the repository settings.
3. Enable GitHub Pages for the desired branch.
4. Use the generated site URL or a custom domain if needed.
